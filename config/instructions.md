Dear Applicant,

As part of your application for the Professorship of Social Psychology at TU Dresden, the selection committee requires you to submit a RESQUE profile. You can use this RESQUE app to create your profile. Please note the following information:

- You may enter as many of your publications into the tool as you wish. However, you must submit **exactly five publications** to the selection committee. Therefore, when exporting your data using "Save to file ...", make sure to select exactly five publications.
- Please mark your **three best publications** as "Top Papers".
- Only publications for which you have completed **all required indicators** will be included in the evaluation. You can tell that a publication has been entered completely if it is marked with a green dot in the publication overview (on the left) and if no issues are displayed in the upper-right corner.
- Once you have entered all your publications, use "Save to file ..." to download your data as a `json` file and submit this file together with your other application documents. Please note that the data from the RESQUE tool are not transmitted automatically to the selection committee.

Data protection notice: The application runs locally in your web browser on your computer (and can also be used offline). Your profile data are stored only in your browser. If you enter a DOI for one of your publications, it will be sent to doi.org to retrieve metadata such as the title, publication year, and abstract. For further details, please refer to the <a href="#" onclick="showPrivacyModal(); return false;">Privacy Policy</a>.

If you have any questions or experience any problems with the RESQUE application, please contact [Dr. Ulrike Senftleben](mailto:ulrike.senftleben@tu-dresden.de).

On the next page, you will find an introduction explaining how to use the RESQUE app.





Liebe Bewerberin, lieber Bewerber,

für die Bewerbung auf die Professur Sozialpsychologie der Technischen Universität Dresden fordert die Kommission ein RESQUE Profil an. Mit dieser App können Sie das Profil erstellen.
Folgende Informationen sind wichtig:

- Sie können beliebig viele ihrer Publikationen in das Tool eintragen, aber **genau 5 Publikationen** bei der Kommission einreichen.
  → Wenn Sie die Daten mit "Save to file ..." exportieren, wählen Sie also genau 5 Publikationen aus.
- Bitte markieren Sie ihre **3 besten Publikationen** als "Top Papers".
- Es werden nur Publikationen ausgewertet, bei denen Sie auch die Indikatoren **vollständig ausgefüllt** haben. Ob eine Publikation vollständig eingetragen ist, erkennen Sie an einem grünen Punkt neben der Publikationen in der Übersicht (links), und dass rechts oben keine Probleme angezeigt werden.
- Wenn Sie alle Publikationen eingeben haben, laden Sie mit "Save to file ..." die Daten als `json`-Datei herunter und reichen Sie diese zusammen mit ihren anderen Unterlagen ein. Die Daten aus dem RESQUE Tool werden *nicht* automatisch an die Kommission übertragen.

Hinweis zum Datenschutz: Die App läuft lokal im Browser auf Ihrem Rechner (auch offline) und Ihre Profildaten werden nur in Ihrem Browser gespeichert. Wenn Sie dois Ihrer Paper eingeben, werden diese an doi.org übermittelt um Metadaten wie Titel, Jahr und Abstract abzurufen. Für Details, siehe die <a href="#" onclick="showPrivacyModal(); return false;">Datenschutzerklärung</a>.
