Dear Applicant,

As part of your application for the **Professorship of Social Psychology** at TU Dresden, the selection committee requires you to submit a RESQUE profile. You can use this RESQUE app to create your profile. Please note the following information:

- You may enter as many of your publications into the tool as you wish. However, you must submit **exactly 5 publications** to the selection committee. Therefore, when exporting your data using "Save to file ...", make sure to select exactly five publications.
- Please mark your **3 best publications** as "Top Papers".
- Only publications for which you have completed **all required indicators** will be included in the evaluation. You can tell that a publication has been entered completely if it is marked with a green dot in the publication overview (on the left) and if no issues are displayed in the upper-right corner.
- Once you have entered all your publications, use "Save to file ..." to download your data as a `json` file and submit this file together with your other application documents [here](https://redcap.zih.tu-dresden.de/redcap/surveys/?s=QVZtXcZzzncykKKa). Please note that the data from the RESQUE tool are not transmitted automatically to the selection committee.

Data protection notice: The application runs locally in your web browser on your computer (and can also be used offline). Your profile data are stored only in your browser. If you enter a DOI for one of your publications, it will be sent to doi.org to retrieve metadata such as the title, publication year, and abstract. For further details, please refer to the <a href="#" onclick="showPrivacyModal(); return false;">Privacy Policy</a>.

If you have any questions or experience any problems with the RESQUE application, please contact [Dr. Ulrike Senftleben](mailto:ulrike.senftleben@tu-dresden.de).
