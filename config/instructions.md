Liebe Bewerberin, lieber Bewerber,

für die Bewerbung auf die Professur Sozialpsychologie der Technischen Universität Dresden fordert die Kommission ein RESQUE Profil an. Mit dieser App können Sie das Profil erstellen.
Folgende Informationen sind wichtig:

- Sie können beliebig viele ihrer Publikationen in das Tool eintragen, aber **genau 5 Publikationen** bei der Kommission einreichen.
  → Wenn Sie die Daten mit "Save to file ..." exportieren, wählen Sie also genau 5 Publikationen aus.
- Bitte markieren Sie ihre **3 besten Publikationen** als "Top Papers".
- Es werden nur Publikationen ausgewertet, bei denen Sie auch die Indikatoren **vollständig ausgefüllt** haben. Ob eine Publikation vollständig eingetragen ist, erkennen Sie an einem grünen Punkt neben der Publikationen in der Übersicht (links), und dass rechts oben keine Probleme angezeigt werden.
- Wenn Sie alle Publikationen eingeben haben, laden Sie mit "Save to file ..." die Daten als `json`-Datei herunter und reichen Sie diese zusammen mit ihren anderen Unterlagen ein. Die Daten aus dem RESQUE Tool werden *nicht* automatisch an die Kommission übertragen.

Hinweis zum Datenschutz: Die App läuft lokal im Browser auf Ihrem Rechner (auch offline) und Ihre Profildaten werden nur in Ihrem Browser gespeichert. Wenn Sie dois Ihrer Paper eingeben, werden diese an doi.org übermittelt um Metadaten wie Titel, Jahr und Abstract abzurufen.
